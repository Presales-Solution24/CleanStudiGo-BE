# Studigo - Backend API

Studigo adalah platform edukasi dan informasi untuk produk-produk teknologi (printer, scanner, projector) yang ditujukan untuk sales dan partner.

Proyek ini dibangun menggunakan:

- Python Flask
- Flask-SQLAlchemy
- Flask-Migrate
- MySQL Database
- JWT Authentication
